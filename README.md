# screenshot-lite

一个基于 PySide6 的轻量级截图工具 MVP，支持：

- 区域截图
- 全屏截图
- 简单标注（矩形、箭头、文本、马赛克）
- 保存到本地
- 复制到剪贴板
- 将截图固定为置顶贴图
- 系统托盘常驻
- 选区确认态：拖动、调整大小、双击确认或按钮确认
- 贴图直接编辑：矩形、箭头、文本、马赛克直接作用在贴图上
- 透明悬浮工具栏：选择、标注、撤销、重做、复制、保存、透明度调节、关闭
- 标注颜色调整：矩形、箭头、文本可直接切换颜色
- OCR 文字识别：对当前截图或贴图识别文字并可复制结果
- OCR 结果窗口：浅色可读界面，支持直接翻译与复制翻译结果
- 选择模式缩放：贴图支持四角与四边共八向缩放手柄
- 托盘快捷键设置：支持修改区域截图与全屏截图的全局热键
- 托盘翻译设置：支持设置源语言、目标语言和 OCR 后自动翻译
- OCR 图像预处理：自动执行放大、增强对比度、二值化和暗色主题适配，提高代码块与中文识别率
- 截图确认态优化：增加尺寸提示、快捷操作提示和更清晰的确认栏

## 环境要求

- Python 3.12+
- Linux、Windows、macOS
- OCR 依赖系统已安装 tesseract
- 在线翻译依赖网络访问能力

## 近期交互优化

- 贴图在选择模式下可直接拖动，也可通过八向手柄缩放，交互更接近常见贴图工具。
- 截图确认态会显示选区尺寸，并保留 Enter、双击确认和 Esc 取消等快捷操作。
- OCR 会针对普通截图、代码块和暗色背景分别尝试不同预处理版本，再选择识别结果更好的文本。

## 安装

```bash
python3 -m venv .venv
. .venv/bin/activate
python -m pip install -r requirements.txt
```

如果工作区已经存在 .venv，可直接激活后运行。

## 运行

```bash
. .venv/bin/activate
python main.py
```

或直接执行：

```bash
./run.sh
```

## 默认操作

- 托盘菜单：区域截图、全屏截图、快捷键设置、退出
- 区域截图后：先进入确认态，可拖动选区、拉伸边框，双击或点击“确认”后直接变成贴图
- 贴图侧边透明工具栏：选择、矩形、箭头、文本、马赛克、撤销、重做、复制、保存、透明度调节、关闭
- 工具栏现在改为顶部居中的透明浮动面板，减少对截图内容的遮挡
- 工具栏提供颜色按钮和预设色块，可直接调整矩形、箭头、文本颜色
- `识别文字`：调用 tesseract 对当前图像做 OCR，并弹出结果窗口
- 托盘中可设置区域截图与全屏截图快捷键，保存后会立即尝试注册全局热键
- `Ctrl+C`：复制到剪贴板
- `Ctrl+S`：保存图片
- `Ctrl+P`：固定为贴图
- `Esc`：关闭当前窗口

## 说明

这是一个可运行的 MVP，优先实现了功能清单里的核心链路。当前已经支持快捷键配置持久化；窗口截图、更多贴图交互和跨桌面环境的全局热键兼容性仍可继续增强。
